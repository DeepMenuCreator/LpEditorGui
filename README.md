# LPGuiEditor

GUI-редактор прав LuckPerms прямо в игре: смотреть и редактировать права игроков
и групп, добавлять/удалять узлы прав, добавлять игрока в группу — без консоли.

## Сборка

Нужен Maven и JDK 21.

```bash
cd LPGuiEditor
mvn clean package
```

Готовый файл появится в `target/LPGuiEditor.jar`.

## Установка

1. Убедись, что на сервере уже стоит **LuckPerms** — плагин не запустится без него.
2. Положи `LPGuiEditor.jar` в папку `plugins/`.
3. Перезапусти сервер.

## Использование

Команда: `/lpgui` (право `lpgui.use`, по умолчанию — у оператора).

Открывается главное меню:

- **Права игроков** → список онлайн-игроков (или ввод ника вручную в чат для
  оффлайн-игрока) → меню с его правами.
- **Права групп** → список групп LuckPerms → меню с правами группы.

В меню прав:
- **ЛКМ по праву** — показать его полностью в чат.
- **Shift+ЛКМ по праву** — удалить это право (сразу, без подтверждения).
- **Добавить право** — открывает чат-приглашение. Пиши:
  - `some.permission.node` — выдать право (true)
  - `-some.permission.node` — выдать право со значением false
  - `отмена` — отменить ввод
- **Добавить в группу** (только у игрока) — открывает список групп, клик по
  группе добавляет игрока в неё.

## Структура проекта

```
src/main/java/com/mystplugin/lpgui/
 ├── LPGuiEditor.java           — главный класс, регистрация команд/слушателей
 ├── gui/
 │   ├── GuiManager.java        — построение всех меню (Inventory)
 │   └── Holders.java           — InventoryHolder-классы для распознавания меню
 └── listener/
     ├── GuiListener.java       — обработка кликов по GUI
     └── ChatInputListener.java — обработка ввода прав/ников через чат
```

## Заметки

- Для оффлайн-игроков используется `LuckPerms#getUserManager().lookupUniqueId(name)`,
  так что найдутся только те, кто хотя бы раз заходил на сервер.
- Изменения сохраняются сразу через `saveUser`/`saveGroup`, вступают в силу
  без перезахода (LuckPerms сам применяет их через свой event bus).
- Если прав/групп много — работает постраничная навигация (45 слотов на страницу).
